# Cookies

Cookies is an umbrella repository for smaller projects that I do not plan to update frequently. Each subfolder used to be its own repository on GitHub, but I consolidated them into this one repository here.

## Projects

- [📁 **Main**](/0-main/) – Smallers projects in functions in Python language.
- [📁 **C++ Useful Notes**](/1-cpp-useful-notes/) – My personal notes for the C++ language.
- [📁 **Word Count**](/2-word-count/) – Count the number of words in a text file in C++ language.
- [📁 **Basic Arithmetic**](/3-basic-arithmetic/) – Cheatsheet for using arithmetic operations in Assembly language.
- [📁 **Swanky Rebels**](/4-swanky-rebels/) – Text adventure game in Batch language.
- [📁 **Sequential Sum**](/5-sequential-sum/) – Demonstrates run time complexity in Python language.
- [📁 **Lexical Links**](/6-lexical-links/) – A word association game in JavaScript language.
